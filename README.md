# HR-Attrition-Data

An HR analytics project analyzing employee attrition patterns and visualizing key workforce insights using Tableau.

## Overview

This repository contains data and analysis related to employee attrition within an organization. The project identifies key factors contributing to employee turnover and presents findings through interactive Tableau dashboards and written reports.

## Repository Contents

- `HR_Attrition.csv` — Main dataset containing employee records and attrition labels
- - `map_coordinates.csv` — Geographic coordinate data used for map visualizations
  - - `HR_Attrition-Project.twbx` — Tableau workbook with interactive dashboards
    - - `Data Narrative.pdf` — Detailed narrative explaining the data and analytical approach
      - - `Executive Summary (for Leadership Team).pdf` — High-level summary of findings for stakeholders
       
        - ## Key Insights
       
        - The analysis explores attrition drivers such as department, job role, salary, work-life balance, and tenure to help organizations make data-informed HR decisions.
       
        - ## Tools Used
       
        - - Tableau (data visualization)
          - - Excel / CSV (data management)
